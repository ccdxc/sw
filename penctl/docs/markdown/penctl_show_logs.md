## penctl show logs

Show logs from Naples

### Synopsis



------------------------------
 Show Module Logs From Naples 
------------------------------


```
penctl show logs [flags]
```

### Options

```
  -h, --help            help for logs
  -m, --module string   Module to show logs for
			Valid modules are:
				nmd
				netagent
				tmagent
				pciemgrd

```

```
  -y, --yaml               display in yaml format
```
